Field use

Default assumptions

Dry summers, harsh sun, rocky soils, browsing pressure, and not much pampering after planting. In other words, this is not a spa resort for thirsty divas.

Best bets

Lean, well-drained, drought-tolerant plants near camp; natives and tough shrubs farther out; stream-adjacent pockets reserved for species that appreciate slightly more moisture.

Frost reality

Use late-spring timing carefully and treat warm-climate experiments with suspicion. Star jasmine gets a clipboard, a probationary badge, and zero promises.

Site framework

This guide assumes the campground sits on the Knot Tablelands in Klamath County, with zone 6b used as the default planning label, strong sun, dry summers, deer and pronghorn pressure, ponderosa/aspen patterns, and a recovering landscape shaped by the Bootleg fire. Use microclimates, structures, and moisture pockets to break ties when a plant seems borderline.

What’s in here

Each plant card includes zone, toxicity for cats, dogs, and humans, care notes, planting instructions, ideal property location, and a verified external link when I could find a solid regional or local source. Already-planted plants are marked separately from recommended additions so you can tell what is established versus what is still garden propaganda.

How to edit later

This file is designed so Perplexity or a human can update it without digging through a spaghetti swamp.

  • Edit PLANT_DATA to add, remove, or revise plants.
  • Edit SITE_NOTES_DEFAULT to change the starting notes text.
  • Edit the SVG labels in the map section if the camp layout evolves.
  • Keep verified external links only; if a link dies, remove it instead of inventing a replacement.

GitHub upload

  1. Upload this file as index.html at the root of your GitHub Pages repo.
  2. Commit and push to GitHub.
  3. In GitHub, open Settings → Pages, choose the branch, and publish from the repo root.
  4. Open the published site on iPhone or iPad and test filters, notes, and links.

Link rules used here

External links open in a new tab and point to regional or plant-reference pages only when I could supply a real URL. Local and regional sources were preferred where available, especially Portland Nursery and Xera.
